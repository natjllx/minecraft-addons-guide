# O que é JSON?

O json (que é uma abreviação de JavaScript Object Notation) é um formato de arquivo que armazena informações estruturadas em pares de chave e valor.

## Exemplo:

```json
{
  "nome": "Natjllx",
  "numero": "10.0",
  "ativo": true
}
```

## Estrutura:

- `{}` - é um objeto
- `"chave": valor` - pares
- Os valores podem ser texto, números, booleanos, listas

## No contexto do minecraft

No minecraft o JSON é usado em quase tudo praticamente:

- **manifest.json** - informações do pacotes
- **Blocos customizados** - propriedades dos blocos
- **Itens** - configurações de itens
- **Entidades** - comportamento dos mobs
- **Loot tables** - itens que caem
