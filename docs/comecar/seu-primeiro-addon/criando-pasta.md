# Modificando a Terra

Iremos criar nosso primeiro addon para entender melhor como tudo funciona na prática.

## Passo 1: Criar a Pasta do Addon

1. Abra a pasta **development_resource_packs**
2. Crie uma nova pasta com um nome descritivo, por exemplo: **primeiro_addon_rp** (rp significa Resource Pack)
3. Dentro dessa pasta crie outra pasta chamada **textures** e dentro dela crie a pasta **blocks**, ou seja tem que ficar assim: meu_addon_rp/textures/blocks/.
4. Na raiz, se quiser, pode colocar um ícone do addon como se fosse uma capa, com o nome **pack_icon.png**, o tamaho recomendado para a resolução é 256x256 ou 128x128, 64x64 também funciona.
5. Depois, na raiz crie um arquivo chamado **manifest.json**
