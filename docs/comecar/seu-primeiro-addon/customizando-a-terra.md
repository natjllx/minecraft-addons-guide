# Customizando a Terra

Como vamos apenas fazer uma demonstração, vou mostrar como criar uma terra vermelha simples.

## Passo 1: Preparar a Imagem

1. Abra um software de desenho (Photoshop, GIMP, Paint.NET, Krita, ou qualquer outro)
2. Crie uma nova imagem quadrada com uma das seguintes resoluções:
   - **64x64 pixels** (recomendado)
   - **32x32 pixels** (também funciona)

## Passo 2: Desenhar a Textura

Neste exemplo, vamos desenhar uma terra vermelha simples. Você pode:
- Preencher toda a imagem com a cor vermelha
- Adicionar variações de cor para dar mais realismo
- Criar padrões ou detalhes

## Passo 3: Salvar a Imagem

Salve a imagem com o nome **exatamente assim:** `dirt.png`

**Caminho correto:**
```
meu_primeiro_addon_rp/textures/blocks/dirt.png
```

**Importante:** O nome deve ser `dirt.png` (em minúsculas) e no formato PNG.

## Próximo Passo

Depois de salvar, você poderá ativar o addon no Minecraft e ver sua textura de terra vermelha funcionando!
